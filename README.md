# Garden of Teeth

Garden of Teeth is a relaxing creature-garden roguelite where the player tends a living inventory.

## Project

- Engine: Godot 4.x
- Current run scene: `res://game/scenes/debug/first_fun_test.tscn`
- Current MVP goal: Prove Lantern Lily -> Light -> Saintmoth -> Shield.

## Controls

- WASD: move
- Space: pulse

## Development Rule

Do not add new content until the smallest fun loop is playable.

## Opening The Project

1. Open Godot 4.x.
2. Import or open this repository folder.
3. Run the current main scene, `res://game/scenes/debug/first_fun_test.tscn`.
4. Use WASD to move and Space to Pulse Saintmoth once Lantern Lily has produced Light.
